# 🛡️ Phishing URL Detector

A Python-based GUI application that detects phishing URLs using rule-based analysis.

## 🚀 Features
- Detects suspicious URLs
- Checks for phishing patterns (IP address, keywords, HTTP usage)
- GUI built using Tkinter

## 🛠️ Technologies Used
- Python
- Tkinter
- Validators library

## ▶️ How to Run

1. Install dependencies:
pip install validators

2. Run the program:
python phishing_detector.py
